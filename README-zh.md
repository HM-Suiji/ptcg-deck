# 项目介绍

本项目是一个在线ptcg（宝可梦集换式卡牌）对战项目，你可以随时随地来一盘轻松的Ptcg对战。

# 项目预览

# 功能特点

- 🎮 支持在线对战和本地对战模式
- 🃏 完整的卡牌管理系统
- 🎯 直观的游戏界面
- 🌐 跨平台支持，随时随地进行对战

# 项目技术栈

- 语言：TypeScript
- 框架：Next.js + React
- UI：Tailwind CSS + DaisyUI
- 状态管理：jotai

# 项目运行

1. 克隆项目

   ```bash
   git clone https://github.com/HM-Suiji/ptcg-deck.git
   cd ptcg-deck
   ```

2. 安装依赖

   ```bash
   npm install
   # or
   yarn
   # or
   pnpm install
   # or
   bun install # 建议
   ```

3. 运行项目

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

4. 打开浏览器访问 <http://localhost:3000>
